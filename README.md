# Customer-Churn-Prediction-Model

## Project Overview
This project focuses on the development of a predictive classification model designed to identify customers at high risk of churning. By proactively identifying these individuals, businesses can implement targeted retention strategies to maintain their customer base and optimize lifetime value.

## Tech Stack & Tools
- **R**: The core statistical programming language used for data manipulation and modeling.
- **Radiant**: A platform-independent browser-based interface for business analytics in R, used for exploratory data analysis and model building.
- **Classification Algorithms**: Employed various machine learning techniques to achieve high predictive accuracy.

## Methodology
The project, completed in 2026, followed a comprehensive data science lifecycle:
1. **Data Preprocessing**: Cleaning and preparing customer demographic and behavioral data for analysis.
2. **Exploratory Data Analysis (EDA)**: Utilizing Radiant to identify key features and correlations related to customer churn.
3. **Feature Engineering**: Developing new variables to capture complex customer interactions and trends.
4. **Model Selection & Training**: Comparing multiple classification models to find the optimal balance of precision and recall.
5. **Evaluation**: Assessing model performance using metrics such as AUC-ROC and confusion matrices to ensure reliability in a business context.
